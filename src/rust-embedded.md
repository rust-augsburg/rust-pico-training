# Rust Embedded

## Challenges

* `no_std`-Environment
* Cross-Compilation
* Debugging
* Logging

## Possibilities in Rust

 * `rp-pico`
   * easy to learn
   * collection of crates
 * `embassy` 
   * async-based 
   * steeper learning curve
   * support for Wi-Fi and internal LED (on Pico W board)
